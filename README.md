# SampleDelphiApp

SampleDelphiApp is a legacy application originally developed using **Delphi (Object Pascal)**.  
This repository contains the original source code base prior to modernization and migration to .NET.

## Project Overview
This project represents the foundational implementation of the application's business logic, data handling, and UI structure as written in Delphi. 
It serves as a reference point for future migration efforts and helps maintain historical context for the system.

## Tech Stack
- **Language:** Delphi / Object Pascal  
- **Architecture:** Traditional VCL‑based desktop application  

## Purpose of This Repository
- Preserve the original legacy Delphi codebase  
- Provide reference material during the migration to modern .NET technologies  
- Track differences between the original code and the migrated version  

## Repository Structure
- `/src` – Main Delphi source files (`.pas`, `.dfm`, `.dpr`)
- `/resources` – Icons, images, and other application assets
- `/docs` – Any documentation related to the legacy application

## Migration
This codebase **will be migrated to a new C#, .NET WPF application**, located in the repository:  
👉 [SampleCSharpWPF](../SampleCSharpWPF)

The migration will focus on:
- Converting Delphi logic to C#  
- Redesigning UI using WPF  
- Modernizing architecture and patterns
